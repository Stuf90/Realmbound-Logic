# ROYAL INQUEST — RULES

> AGENT FILE. CAVE SPEAK. HUMAN VERSION: `rules.human.md`.

ROYAL INQUEST = SPATIAL MURDOKU-STYLE PLACEMENT PUZZLE. PLACE EVERY CHARACTER ON BOARD OF
CHAMBERS SO ALL CLUES HOLD SAME TIME. THIS DOC = CANONICAL **GAME RULES** — HOW FINISH
CASE PLAY + JUDGE. NOT AUTHOR NEW CONTENT — THAT IN SUB-DOCS BOTTOM.

## BOARD

- GRID `rows x columns` CELLS (`InquestDefinition.rows/columns`).
- EVERY CELL BELONG ONE **CHAMBER** (`chamberId`) — NAME ROOM, VISIBLE BOUNDARY + LABEL.
  CHAMBERS GROUP CELLS, NOT SEPARATE GRID.
- SOME CELLS **BLOCKED** — IMPASSABLE SCENERY (PROP OR OTHER). NO CHARACTER EVER OCCUPY.
- SOME CELLS CARRY **LEGAL-CHARACTER RESTRICTION** (`legalCharacterIds`) — ONLY LISTED
  CHARACTER(S) MAY USE. CELL NO RESTRICTION = LEGAL FOR ANY.

## CHARACTERS

- CASE HAVE FIXED CAST, EACH NAME + PORTRAIT.
- EXACTLY ONE = **VICTIM**.
- EXACTLY ONE = **TRAITOR**, DECIDE BY WHERE EVERYONE END UP (SEE "VICTIM + TRAITOR"
  BELOW) — NOT AUTHOR FACT PLAYER PLACE DIRECT.

## PLACEMENT RULES

FINISH PLACEMENT VALID ONLY WHEN **ALL** HOLD:

1. **ONE CHARACTER PER ROW.** NO TWO SHARE ROW.
2. **ONE CHARACTER PER COLUMN.** NO TWO SHARE COLUMN.
3. **LEGAL CELL ONLY.** CHARACTER USE UNBLOCKED CELL IT PERMIT (NO RESTRICTION, OR ON
   RESTRICTION LIST).
4. **EVERY CLUE SATISFIED.** SEE "CLUES" BELOW.
5. **VICTIM/TRAITOR CONDITION HOLD** (SEE BELOW).

ROWS/COLUMNS INCREASE TOP-TO-BOTTOM, LEFT-TO-RIGHT. 0-INDEX INTERNAL, 1-INDEX IN CLUE
TEXT (ROW INDEX `0` = "FIRST ROW").

## CHAMBERS + ADJACENCY

- **SAME CHAMBER** — TWO CHARACTERS SAME `chamberId`.
- **DIFFERENT CHAMBER** — OPPOSITE.
- **BESIDE** — TWO CHARACTERS ORTHOGONAL ADJACENT (MANHATTAN DIST 1) **+** SAME CHAMBER.
  ADJACENT ACROSS CHAMBER BOUNDARY NOT COUNT.
- **NOT BESIDE** — NEGATE "BESIDE": ANY PLACEMENT NOT BOTH ADJACENT + SAME-CHAMBER.

## CLUES

EVERY CLUE = STRUCTURED PREDICATE, NOT FREE TEXT — SENTENCE SHOW PLAYER = FLAVOR ONLY,
NEVER PARSE FOR CORRECTNESS. SEE
[`authoring/clues-and-predicates.cave.md`](authoring/clues-and-predicates.cave.md) FULL
PREDICATE LIST + EXACT SEMANTICS (EXACT ROW/COLUMN/CHAMBER, SAME/DIFFERENT CHAMBER,
CARDINAL DIRECTION, BESIDE/NOT-BESIDE).

CLUE STATE WHILE PLACE STILL GO:

- **SATISFIED** — TRUE GIVEN CURRENT (MAYBE PARTIAL) PLACEMENT.
- **VIOLATED** — DEFINITE FALSE ALREADY, NO MATTER WHAT UNPLACED STILL.
- **UNDETERMINED** — CAN'T DECIDE YET, NEED CHARACTER NOT PLACED.

ONLY DEFINITE VIOLATION REPORT BEFORE COMPLETE — PLAYER NEVER TOLD WRONG WHEN MERELY
STILL UNKNOWN.

## VICTIM + TRAITOR

TRAITOR NOT INDEPENDENT PLACEMENT FACT — FALL OUT FROM EVERYONE ELSE POSITION:

> TRAITOR = ONLY NON-VICTIM CHARACTER SHARE VICTIM'S CHAMBER, WHEN NO THIRD CHARACTER
> ALSO PRESENT THERE.

MEAN: AT COMPLETE, VICTIM CHAMBER MUST HOLD EXACTLY TWO SOLVE OCCUPANT — VICTIM +
TRAITOR — NO ONE ELSE.

## COMPLETION

CASE COMPLETE ONLY WHEN:

- ALL CHARACTERS LEGAL PLACED (RULE 3);
- ROW + COLUMN UNIQUE HOLD (RULE 1-2);
- EVERY CLUE SATISFIED (RULE 4);
- VICTIM/TRAITOR CONDITION HOLD (RULE 5);
- EVERY CHARACTER SIT ON PUZZLE'S AUTHOR SOLUTION CELL.

LAST POINT MATTER EVEN THOUGH STRUCTURAL RULES ABOVE CHECK INDEPENDENT — CASE HAVE
EXACTLY ONE AUTHOR SOLUTION, COMPLETE NEED REACH IT, NOT MERELY SATISFY EVERY RULE
ABSTRACT (PUZZLE AUTHOR SO THOSE COINCIDE, BUT IMPL CHECK BOTH).

## PLAYER TOOLS

- **PLACE** — PUT SELECT CHARACTER ON LEGAL CELL.
- **CROSS** — MANUAL MARK CELL IMPOSSIBLE, PLAYER OWN BOOKKEEP. INDEPENDENT FROM GAME
  DERIVE EXCLUSION (CELLS GAME ALREADY KNOW UNAVAILABLE FROM BLOCKED/LEGAL-CELL/ROW/
  COLUMN CONFLICT) — NEVER AUTO-CLEAR WHEN DERIVE EXCLUSION CHANGE.
- **CHECK PROGRESS** — REPORT FIRST DEFINITE PROBLEM ONLY (ILLEGAL/BLOCKED PLACEMENT,
  DUPLICATE ROW, DUPLICATE COLUMN, VIOLATE CLUE, CHARACTER NO LEGAL CELL LEFT, INVALID
  VICTIM/TRAITOR CHAMBER) — NO REVEAL UNRELATED CORRECT DESTINATION.
- **HINT** — REPORT EXIST CONTRADICTION FIRST; ELSE OFFER ONE DETERMINISTIC DEDUCTION +
  REASON, OPTION APPLY.
